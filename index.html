Ringkasan Perilaku API

Native PHP PDO JSON Response Postman Friendly Tanpa Framework
Endpoint dibuat per tabel. Jika ingin mengambil data spesifik, tambahkan ?id=1. Untuk request body, gunakan JSON dengan nama field yang sudah dipetakan di dokumentasi ini.
Catatan penting: karena struktur database kamu memakai nama kolom yang mengandung spasi, backend sudah memetakan nama field JSON agar lebih nyaman dipakai saat request.

Format Response Umum

{
  "status": "success",
  "message": "Data berhasil diambil",
  "data": [...]
}
Nilai status biasanya success atau error. Field data berisi array data, satu objek, atau null tergantung hasil request.

Contoh Hasil Request

Hasil GET

{
  "status": "success",
  "message": "Data berhasil diambil",
  "data": [
    {
      "nama_sekolah": "SMA Negeri 1 Makassar",
      "id_sekolah": 1
    }
  ]
}

Hasil POST

{
  "status": "success",
  "message": "Data berhasil ditambahkan",
  "data": {
    "id_sekolah": "2001"
  }
}
Contoh efek di database setelah POST: data baru muncul di tabel dengan ID auto increment baru.

Hasil PUT

{
  "status": "success",
  "message": "Data berhasil diupdate",
  "data": {
    "id_sekolah": 2001
  }
}
Contoh efek di database setelah PUT: baris dengan ID yang sama akan berubah isi kolomnya, sedangkan ID tetap.

Hasil DELETE

{
  "status": "success",
  "message": "Data berhasil dihapus",
  "data": {
    "id_sekolah": 2001
  }
}
Contoh efek di database setelah DELETE: baris dengan ID tersebut dihapus dari tabel.

Catatan Penting per Endpoint

Endpoint Field yang dapat diubah Field yang tidak dapat diubah Contoh Body JSON
/dim_jenis_kelamin l_p id_jenis_kelamin {"l_p":"P"}
/dim_kabupaten_kota kabupaten_kota id_kabupaten_kota {"kabupaten_kota":"Gowa"}
/dim_keterangan ket id_keterangan {"ket":"Lulus"}
/dim_lama_studi lama_studi id_lama_studi {"lama_studi":"4 Tahun"}
/dim_sekolah nama_sekolah id_sekolah {"nama_sekolah":"SMK Negeri 1"}
Karena primary key bersifat auto increment, nilainya tidak perlu dikirim saat POST. Untuk PUT dan DELETE, ID dikirim lewat query string ?id=....
